## Installation Guide

The installation of pygame is required to run this game, highlighted in official 
documentation from 'https://www.pygame.org/wiki/GettingStarted'.

Further instructions specific to operating systems can be found from the following 
installation guides:

Mac:  	 https://www.pygame.org/wiki/GettingStarted#Mac%20installation

Unix:	 https://www.pygame.org/wiki/GettingStarted#Unix%20Binary%20Packages

Windows: https://docs.python.org/3/using/windows.html#installation-steps


Furthermore, python3 installation resources can be found at 'https://docs.python.org/3/'


In order to execute this file:

1) Unzip the contents of 'ctrl-alt-elite.zip' into a directory of your choice
2) Load this into the python IDE of your choice
3) Run the main.py file
